## Installation

1.  Install DSL React Table as a dependency

```bash
# Yarn
$ yarn add dsl-react-table

# NPM
$ npm install dsl-react-table
```

2.  Import the `dsl-react-table` module

```javascript
import { dslReactTable } from 'dsl-react-table'
```
